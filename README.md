# random-image
An api that gets random images from the image folder
 Upload images to the folder named images. Accepted formates:- png, jpg, jpeg.
 deploy this on any server!
### Deploy on Render
1. Fork Repo
2. Upload your images in `./images`
3. Goto Render.com
   
   [![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=./)
